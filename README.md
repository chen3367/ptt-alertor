# Ptt Alertor
## 指令
`!help`</br>
指令查詢

`!ptt <版名> <關鍵字> <頁數>`</br>
手動搜尋文章(頁數預設三頁)

`!add <版名> <關鍵字>`</br>
新增自動搜尋清單

`!delete <版名> <關鍵字>`</br>
刪除自動搜尋清單

`!deleteall <版名>`</br>
刪除該版所有自動搜尋關鍵字

`!list`</br>
列出自動搜尋清單
